1. Spring Web

Spring Web is a module in Spring Boot that helps to build web applications.
It allows you to create APIs, handle requests of client (GET, POST, etc.), and send JSON responses.
It helps to create REST APIs using Spring MVC.
It uses DispatcherServlet as the front controller.
It contains annotations like:
•	@RestController
•	@GetMapping
•	@PostMapping
•	@RequestBody
•	@PathVariable

2. MySQL Driver

MySQL driver is a small software that helps Spring Boot connect to the MySQL database.
When you add this dependency:
mysql-connector-j
Spring Boot understands how to:
•	connect to MySQL
•	run queries
•	read and write data
•

3. Spring Data JPA

Spring Data JPA helps you interact with the database easily without writing long SQL queries. It automatically handles CRUD operations.
Spring Data JPA follows the Repository Pattern and reduces boilerplate code. 
Example:
findByEmail() → Spring creates query:
SELECT * FROM user WHERE email = ?
It uses Hibernate internally.

4. Lombok

Lombok is a tool that reduces boilerplate code like getters, setters, constructors, toString etc.

5. ORM (Object Relational Mapping)

ORM is a technique that maps Java classes to database tables. Or ORM is a concept in which java class connect to the database table.

6. ORM Tool

Tools that implement ORM concept are called ORM tools.
Example: Hibernate.

7. Hibernate

Hibernate is the most popular ORM tool.
It maps Java objects to database tables and handles queries automatically.
It handles:
•	table creation
•	data saving
•	queries
•	relationships (OneToMany, ManyToOne, etc.)
•	caching

8. Entity

An Entity is a Java class that represents a table in the database.
Entity class contains @Id which is used as primary key.

9. Controller

Controller receives the request from the user and returns a response.
It is the entry point for APIs.

10. Service

Service contains business logic.
It handles calculations, processing and main operations.

11. Repository

Repository interacts with the database.
It performs CRUD operations using JPA.
•  save()
•  findAll()
•  findById()
•  delete()
Repository extends JpaRepository or CrudRepository.
pring creates runtime proxy using JDK Dynamic Proxy or CGLIB.

12. IOC (Inversion of Control)

IOC means instead of you creating objects, Spring creates and manages objects.
IoC means Spring controls the creation of objects instead of you creating them with new.

13. Dependency Injection (DI)

DI is a technique where Spring provides the required object to another object automatically.
Dependency Injection means Spring provides required objects (dependencies) to your class automatically.

14. Constructor Injection

Constructor injection means Spring puts (injects) dependencies through the constructor of the class.
