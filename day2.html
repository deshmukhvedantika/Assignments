Function and their types

Function: A function is block of code that perform specifice task and it can be executed when it is called and it is a reusable .

1 Function declaration (named function)

  • Function declaration means a function which has name.
  • Start with function keyword followed by function name.
  • Function written anywhere (below or above )you can called due to hoist.
  • Example: 
      
			    function greet() {
  			        console.log("Hello");
                    	    }
                   	    greet();

    • 2 Function Expression

  • A function assigned to the variable.
  • No need to gives function name (Fine if gives or not).
  • Cannot be hoist so you must have to declare before call.
  • used for anonymous function,callback function, if we want to use variable as function.
  • Example: 
    			 let greet = function() {
 			 	console.log("Hello");
            		 };
           		 greet();

    • 3 Arrow Function (=> function)

  • short and modern technique to write a function.
  • There is no need of write function keyword meanwhile use =>(arrow).
  • useful for callback function.
  • Example: 
          
			let greet = () => {
  				console.log("Hello");
            		};
            		greet();
  • Shorter version: if you want to write in single line then - no need to write {} and return.
    let add = (a, b) => a + b;

    • 5. Callback Function

  • Callback function means one function gives to another function as argument and then call it. (function inside function)
  • used when any work would be do later or after complition.
  • Example:
    	function greet(name) {
  		console.log("Hello " + name);
	}

	function caller(callback) {
  		callback("Vedantika");
	}
	caller(greet);

here greet is the callback which used in another function.

  •  Real-life example 
setTimeout(function() {
  console.log("This runs after 2 seconds");
}, 2000);
Here function() { ... } is callback-works after 2 seconds.

    • 6. Constructor Function.

  • used to create object (same type)
  • this function name always starts with Capital letter.
  • use new keyword to create object.
  • Example: 
	function Person(name, age) {
  	this.name = name;
  this.age = age;
}

let user1 = new Person("Vedantika", 20);
let user2 = new Person("Aarav", 25);

console.log(user1.name);  // Vedantika
console.log(user2.age);   // 25

    • 7. Higher-Order Function (HOF)

  • A Higher-Order Function is a function that either:
1.	takes another function as an argument,
or
2.	returns another function.
A function that works with other functions.

  •     Example 1 – Takes another function as an argument
function greet() {
  console.log("Hello");
}

function caller(func) {
  func();  // calling greet
}

caller(greet);
Here, caller() is a Higher-Order Function
because it accepts another function (greet) as a parameter.

  •     Example 2 – Returns another function
function outer() {
  return function() {
    console.log("Inside function");
  };
}

let innerFunc = outer();
innerFunc();
Here, outer() is a Higher-Order Function
because it returns another function.
Used in array methods like map, filter, reduce

    • 8. IIFE (Immediately Invoked Function Expression)

  • An IIFE is a function that runs automatically as soon as it is created. You don’t need to call it separately. It is executed immediately.
  • Looks like: An IIFE is written inside () and then another () is used to call it instantly.
  • Example:
(function () {
  console.log("I run immediately!");
})();

DOM (Document Object Model)

DOM = Document Object Model
When a webpage loads in the browser, the browser reads the HTML and converts it into a tree-like structure called the DOM.
This DOM allows JavaScript to access, read, modify, delete, or add elements on the webpage dynamically.
In simple DOM is a programming interface that represents the webpage as objects, so JavaScript can interact with it.

Why is DOM needed?


Because HTML is static.
Once a page loads, HTML alone cannot change anything.

But JavaScript can change:

  1. Text
  2. CSS style
  3. HTML content
  4. Attributes
  5. Add or remove elements
  6. Respond to user actions (click, input, hover, etc.)

This is all possible because of DOM.

Browser converts it into a tree:
JavaScript can access these elements using DOM methods. 
Document
 └── html
      ├── head
      └── body
           ├── h1
           └── p

    Important DOM Methods

  • getElementById() – Get element by id
  • getElementsByClassName() – Get elements by class
  • getElementsByTagName() – Get elements by tag
  • querySelector() – Returns first matching element
  • querySelectorAll() – Returns all matching elements

    • Node Types (Most Important)

  • Element Node
  • Text Node
  • Attribute Node
  • Comment Node

    • Common DOM Properties

  • innerHTML – Change HTML content
  • innerText – Change text content
  • style – Change CSS
  • .src, .href, etc. – Change attributes

    • Common DOM Actions

  • Create element: document.createElement()
  • Add element: appendChild()
  • Remove element: .remove()
  • Add event: addEventListener()
